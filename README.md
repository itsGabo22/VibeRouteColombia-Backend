# Backend — Vibe Route

Servicio REST desarrollado con **Java 17** y **Spring Boot**.
Incluye módulo de optimización de rutas (OR-Tools) e integración con APIs de mapas.

## Comandos

> Aquí van los comandos para levantar el servicio; rellenar tras inicializar el proyecto con Maven o Gradle.

```bash
# Ejemplo (Maven):
# ./mvnw spring-boot:run -Dspring-boot.run.profiles=dev
```
